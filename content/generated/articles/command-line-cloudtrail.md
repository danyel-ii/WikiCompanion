## Limits and common mistakes

A recurring mistake is to confuse tool output with final truth. Command Line CloudTrail is useful because it sharpens interpretation, but it still depends on context, data quality, and the analyst's framing. Another mistake is to treat it as a complete workflow by itself when the surrounding evidence and follow-up analysis usually matter just as much.
